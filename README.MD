## Getting Started

First, Clone the project :

```bash
git clone https://github.com/siunixdev/pwa-pokemon-stock-app
```

go to project directory and install all dependencies with

```bash
npm install
```

and run the development server:

```bash
npm run dev
# or
yarn dev
```
